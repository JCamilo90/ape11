Comandos Com Git Flow

Iniciando Git Flow

git flow init

git flow feature start nome_funcionalidade

Vai Criar um branch (baseado no develop) e uma checkout nesse branch


Finalizando uma funcionalidade

git flow feature finish nome_funcionalidade

Esse comando irá executar as seguintes ações:

Merge com a branch develop;

Deleta a branch funcionalidade;

Checkout na branch develop;

Finalizar uma funcionalidade, você deve executar esse comando dentro do branch funcionalidade que deseja finalizar

Publicando uma funcionalidade

git flow feature publish nome_funcionalidade

Obter uma funcionalidade

git flow feature pull nome_funcionalidade

Simplesmente um pull de uma branch


Versões / Release

Começar uma nova versão

git flow release start versao

Cria um branch baseado no develop


Publicando uma versão

git flow release publish versao


Finalizar versão

git flow release finish versao


Ele executa várias ações:

Faz merge do branch versao no master;

Faz merge do branch versao e um checkout no develop;

Etiqueta a versão com seu nome;

Remove o branch versao;


>

Criando um hotfix

git flow hotfix start hotfix

Cria um branch baseado no master

O argumento hotfix marca a versão com defeito na produção


>

Finalizar um hotfix

git flow hotfix finish hotfix
