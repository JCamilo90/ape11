Principais Exemplos de Comandos do Git

Configurar GIT

git config –global user.name “Seu nome”


Setar email

git config –global user.email “Seu email”

git config –global user.email “Seu email”


Iniciar GIT

git config --global core.editor vim


Criar Copia do repositório

git clone


Adicionar arquivos especificados de código ao seu repositório

git add seu_arquivo (Arquivos Especificados)

git add * (Todos)


Executar o commit dos arquivos que foram adicionados

git commit -m “seu comentário”


Variações de repositorio

git branch (lista todas as ramificações)

git branch (cria um branch com o nome especificado)

git branch -d (deleta o branch com o nome especificado)


Navegar entre as branch

git checkout


Subir suas modificações para um repositório remoto

git push -u


Marcar uma branch e evitar alteração

git tag -a v1.0.0


Ajuda

git help
